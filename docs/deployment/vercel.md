# Деплой на Vercel

## Цель
Описать базовый процесс деплоя приложения на Vercel.

## Ключевые шаги
1. Подключить репозиторий в Vercel.
2. Указать переменные окружения (строка подключения к БД, секреты авторизации).
3. Убедиться, что команды сборки соответствуют `package.json`.
4. После деплоя проверить основные страницы и API маршруты.

## Связанные модули
- `next.config.ts` — конфигурация Next.js.
- `vercel.json` — настройки окружения Vercel (если используются).
- `docs/deployment/database.md` — подключение БД.

## Исторические материалы
Старые инструкции по деплою в `docs/reports/` могут быть неактуальны — используйте их только как контекст.
