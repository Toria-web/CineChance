# Настройка окружения

## Цель
Быстро развернуть локальную среду для разработки и тестирования CineChance.

## Ключевые шаги
1. Установить зависимости проекта: `npm install`.
2. Создать `.env` на основе примера (если есть) и указать строки подключения к БД.
3. Применить миграции Prisma: `npx prisma migrate deploy` (или `npx prisma migrate dev` для локальной разработки).
4. Запустить приложение: `npm run dev`.

## Проверки
- Запуск линтера: `npm run lint`.
- Проверка типизации: `npm run typecheck` (если команда настроена).

## Связанные модули
- `prisma/` — настройки базы данных и миграции.
- `src/app/` — основной код приложения.
- `scripts/` — вспомогательные скрипты (если используются).

## Исторические материалы
Если в `docs/reports/` встречаются пошаговые инструкции — воспринимайте их как исторические: актуальные команды описаны выше.
